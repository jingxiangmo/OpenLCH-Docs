## Setup

Clone repository:
```bash
git clone --recursive https://github.com/Zeroth-Robotics/OpenLCH
cd OpenLCH
```

Create conda environment:
(install Mini Conda [here](https://docs.conda.io/en/latest/miniconda.html) if you haven't done it yet)

```bash
conda create -n humanoid-os python=3.12
conda activate humanoid-os
```

## Inference

Install dependencies:
```bash
pip install -r experiments/requirements.txt

```
Test if ping Milk-V is working:
```bash
ping 192.168.42.1
```

Run inference:

```bash
python experiments/run.py
```
