# 3D Printing Guide


## Print Settings

Ideally with Bambu printers parameter:

| Setting   | Parameter  | Value       |
|-----------|------------|-------------|
| Process   |            | 0.16mm      |
| Strength  | Wall loops | 5           |
| Support   | Type       | tree (auto) |
