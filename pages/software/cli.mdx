# Introduction

OpenLCH is a Python package for interacting with robot. This guide provides a quick overview of how to use the package.

# Installation

To install OpenLCH, use pip:
```bash
pip install openlch
```

# Usage

To use OpenLCH, you need to have a MilkV board and the OpenLCH package installed.

```python
from openlch import HAL

hal = HAL()

# Get the current position of the servo
positions = hal.servo.get_positions()
print(positions)

# Set the position of the servos
hal.servo.set_positions([1, 2, 3], [90, -90, 13.6])
```
