# Introduction

OpenLCH is a Python package for interacting with robot. This guide provides a quick overview of how to use the cli.

## Installation

To install OpenLCH, use pip:
```bash
pip install openlch
```

## Usage
```bash
openlch --help
```

## Update

```bash
pip install openlch -U
```
